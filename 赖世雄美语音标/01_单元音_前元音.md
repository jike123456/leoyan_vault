# 01_单元音_前元音 (The Front Vowels)

前元音的特点是：**舌尖抵住下齿背，舌面前部向硬腭抬起**。区别在于开口度的大小。

## 1. /i/ (长元音)
-   **赖老师诀窍**：
    -   嘴角用力向两边拉开，像**微笑**一样。
    -   肌肉是**紧张**的 (Tense Vowel)。
    -   发出“一”的音，但要把嘴拉得更扁。
-   **例词**：see, tea, me, key

## 2. /ɪ/ (短元音)
-   **赖老师诀窍**：
    -   **这是中国学生最容易发错的音！** 千万不要发成短的“一”。
    -   它是**松元音** (Lax Vowel)。肌肉完全放松。
    -   介于“一”和“耶”之间。肚子放松，短促地发声。
    -   **口诀**：**“军人站岗是 /i/ (紧)，流浪汉瘫倒是 /ɪ/ (松)。”**
-   **例词**：sit, it, big, list

## 3. /e/ (KK特有，注：有些教材写作 /eɪ/)
-   **赖老师诀窍**：
    -   嘴角咧开，比 /i/ 稍开一点。
    -   这是一个**不滑动**的单元音（在 K.K. 中），但实际发音常带有 /ɪ/ 的尾巴，听起来像 A-B-C 的 A。
-   **例词**：pay, day, say

## 4. /ɛ/ (谢孟媛/赖世雄重点)
-   **赖老师诀窍**：
    -   这就是“耶”的那个口型，但是不发“耶”的音。
    -   开口度：能放进**一个小指头**。
    -   **“Bed” 测试**：如果你把 bed 发成了 bad，说明嘴张太大了；发成了 bid，说明张太小了。
-   **例词**：bed, let, get, pen

## 5. /æ/ (大嘴以，蝴蝶音)
-   **赖老师诀窍**：
    -   **全宇宙嘴巴张最大的音！**
    -   开口度：能竖着放进**三根手指**。
    -   发音时，不仅下巴要掉下来，嘴角还要稍微往两边裂一点。
    -   发音像羊叫：“咩——”。
-   **例词**：bad, cat, map, apple
