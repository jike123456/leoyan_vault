# 05_美语发音独门秘籍 (进阶)

学会了单个音标只是第一步，赖老师的精华在于这些让英语听起来“地道”的规则。

## 1. 浊化 (Flap T)
-   **规则**：当 /t/ 夹在两个**元音**之间，且**重音在前面**时，/t/ 变成快速的轻 /d/。
-   **效果**：听起来更流利，不顿挫。
-   **例子**：
    -   wa**t**er -> wa-der
    -   be**tt**er -> be-der
    -   ci**t**y -> ci-dy

## 2. 吞音 (Stop T)
-   **规则**：当 /t/ 在词尾，或者后面接辅音时，**只做口型，不送气**。
-   **赖老师诀窍**：把气憋在喉咙里。
-   **例子**：
    -   ho**t** dog -> ha- dog (t 不发出来，停顿一下)
    -   ba**t**man -> ba- man

## 3. 连读 (Linking)
-   **辅音 + 元音**：
    -   Ge**t u**p -> Ge-dup
    -   Lo**ok a**t -> Loo-kat
-   **r 连读**：
    -   Far away -> Fa-ra-way (前面的 r 会和后面的元音连起来)

## 4. 略读 (h 省略)
-   **规则**：代词 he, him, her, his 在句子中间弱读时，/h/ 往往不发音。
-   **例子**：
    -   Tell **h**im -> Tell-im
    -   Lik**e h**er -> Li-ker
