# 04_辅音_重要难点突破

赖老师特别强调的几个中国学生最易错的辅音。

## 1. /θ/ 与 /ð/ (咬舌音)
-   **死穴**：中国学生常把 thank 发成 sank，把 this 发成 zis。
-   **赖老师诀窍**：
    -   **伸舌头！伸舌头！伸舌头！**
    -   一定要把舌尖伸出上下齿之间，轻轻咬住（不要真咬，是贴住）。
    -   /θ/ (清)：只吹气，不震动声带。例：**th**ank, **th**ree.
    -   /ð/ (浊)：吹气且震动声带，舌头会麻麻的。例：**th**is, fa**th**er.

## 2. /l/ (Dark L)
-   **死穴**：放在词尾时（如 well, milk），很多人不会发，直接略过。
-   **赖老师诀窍**：
    -   **词首** (Light L)：像中文“乐”。如：**l**ike.
    -   **词尾** (Dark L)：
        1.  先发前面的元音。
        2.  发完后，**慢慢把舌尖向上卷，抵住上齿龈（门牙后面那块肉）**。
        3.  喉咙发出低沉的“欧”音。
        4.  Try: `girl` -> `g-ir-欧-舌尖抵上去`。
-   **例词**：girl, world, school, milk

## 3. /r/ (卷舌音)
-   **赖老师诀窍**：
    -   发音开始前，嘴巴就要撅起来，呈圆形。
    -   舌头卷在口腔中间，**绝对不能碰到上颚**。
    -   Try: `rose` -> 嘴巴先撅圆，再发 rrr-ose。

## 4. /v/ 与 /w/
-   **死穴**：very much 读成 wery much。
-   **赖老师诀窍**：
    -   /v/：**上牙齿轻咬下嘴唇**。要有摩擦感。
    -   /w/：双唇撅圆，不咬嘴唇。
    -   练习：`very well` (咬唇 -> 撅唇)。

## 5. /n/ (鼻音) 在词尾
-   **死穴**：nine 发成“奈”，short n 没了。
-   **赖老师诀窍**：
    -   发完音后，**嘴巴不要闭上，舌尖要贴到上齿龈**，让声音从鼻子里断掉。
    -   Try: `nine`, `sun`, `man`.
