# 03_双元音 (Diphthongs)

双元音的核心要义：**滑动 (Glide)**。从第一个音平滑地滑向第二个音，前长后短，前重后轻。

## 1. /aɪ/
-   **构成**：/ɑ/ -> /ɪ/
-   **赖老师诀窍**：
    -   大嘴巴开始，快速滑向小嘴巴。
    -   一定要把第一个音 /ɑ/ 发足。
-   **例词**：like, sky, high

## 2. /aʊ/
-   **构成**：/ɑ/ -> /ʊ/
-   **赖老师诀窍**：
    -   像被人狠狠掐了一下：“嗷！”
    -   从大嘴巴滑向撅嘴巴。
-   **例词**：house, cow, out

## 3. /ɔɪ/
-   **构成**：/ɔ/ -> /ɪ/
-   **赖老师诀窍**：
    -   从椭圆嘴（鸭蛋嘴）滑向咧嘴笑。
-   **例词**：boy, toy, oil

## 4. /ju/ (注意：这是辅音+元音，但常放一起讲)
-   **赖老师诀窍**：
    -   /j/ 是半元音，发音像“耶”的开头。
    -   紧接着发撅嘴的 /u/。
-   **例词**：cute, use, few
