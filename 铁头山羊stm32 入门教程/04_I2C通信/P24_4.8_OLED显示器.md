# 4.8 [I2C]OLED显示器

## 主要内容
使用 I2C 接口驱动 0.96 寸 OLED 屏幕 (SSD1306 驱动芯片)。

### 硬件连接
*   VCC -> 3.3V
*   GND -> GND
*   SCL -> PB8 (举例)
*   SDA -> PB9 (举例)

### 驱动逻辑
*   SSD1306 支持 I2C 命令。
*   **命令字节**：Co=0, D/C#=0 -> 接下来的字节是命令。
*   **数据字节**：Co=0, D/C#=1 -> 接下来的字节是显存数据 (GDDRAM)。

### 常用功能
1.  **OLED_Init()**: 发送一长串初始化命令 (电荷泵、扫描方向、对比度等)。
2.  **OLED_Clear()**: 清空显存 (写 0)。
3.  **OLED_ShowChar()**: 查字库 (5x8 或 8x16 点阵)，将点阵数据写入 GDDRAM。
4.  **OLED_ShowString()**: 循环调用 ShowChar。
