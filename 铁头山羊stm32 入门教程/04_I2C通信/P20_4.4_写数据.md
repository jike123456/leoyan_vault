# 4.4 [I2C]写数据

## 主要内容
使用硬件 I2C 发送数据的流程。

### 发送流程 (作为主机)
1.  **Start**: `I2C_GenerateSTART(I2C1, ENABLE);` 等待 EV5。
2.  **Send Address(Write)**: `I2C_Send7bitAddress(I2C1, SlaveAddr, I2C_Direction_Transmitter);` 等待 EV6。
3.  **Send Data**: `I2C_SendData(I2C1, Data);` 等待 EV8。
4.  **Stop**: `I2C_GenerateSTOP(I2C1, ENABLE);`

*注意*：硬件 I2C 需要严格检查 Event (EVx) 标志位，否则容易卡死。
