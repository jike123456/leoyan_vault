# 1.1 [准备]安装开发环境

## 主要内容
本节主要介绍进行 STM32 开发所需的软件和硬件环境搭建。

### 核心步骤
1.  **集成开发环境 (IDE) 安装**：
    *   通常推荐安装 **Keil uVision5 (MDK-ARM)** 或 **STM32CubeIDE**。
    *   Keil 是经典的嵌入式开发工具，资料丰富；STM32CubeIDE 是 ST 官方推出的免费 IDE，集成了配置工具。
2.  **器件支持包 (Device Family Pack) 安装**：
    *   如果使用 Keil，需要下载并安装对应芯片系列的 Pack 包（如 STM32F1xx_DFP）。
3.  **驱动安装**：
    *   安装 **ST-Link 驱动**，确保电脑能识别仿真器。
    *   安装 **CH340/CP2102 串口驱动**，用于串口通信和调试打印。
4.  **硬件连接**：
    *   连接 ST-Link 仿真器到 STM32 开发板的 SWD 接口（GND, CLK, DIO, 3.3V）。
    *   连接 USB 转串口模块。

### 验证
*   打开 IDE，编译一个简单的工程。
*   连接硬件，点击下载，观察是否能成功烧录程序。
