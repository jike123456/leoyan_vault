# 1.1 [准备]STM32的基本信息

## 主要内容
介绍 STM32 单片机的基本概念、架构特点及命名规则。

### 什么是 STM32？
*   **ST**：意法半导体 (SGS-Thomson Microelectronics)。
*   **M**：Microcontroller (微控制器)。
*   **32**：32 位 (基于 ARM Cortex-M 内核)。
*   STM32 是高性能、低成本、低功耗的 32 位微控制器系列。

### 芯片命名规则 (以 STM32F103C8T6 为例)
*   **STM32**：产品系列。
*   **F**：产品类型 (F = 通用型, L = 低功耗, H = 高性能)。
*   **103**：产品子系列 (103 = 增强型, 72MHz 主频)。
*   **C**：引脚数 (C = 48 pin, R = 64 pin, V = 100 pin, Z = 144 pin)。
*   **8**：Flash 容量 (8 = 64KB, B = 128KB, C = 256KB, E = 512KB)。
*   **T**：封装类型 (T = LQFP)。
*   **6**：温度范围 (6 = -40~85摄氏度)。

### 核心架构
*   基于 **ARM Cortex-M3/M4/M7** 等内核。
*   拥有丰富的外设：GPIO, USART, I2C, SPI, ADC, Timers, USB, CAN 等。
*   使用 **AHB/APB** 总线结构连接内核与外设。
