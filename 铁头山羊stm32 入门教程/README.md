# 铁头山羊STM32 入门教程

## 简介
本教程是由 B站UP主“铁头山羊”制作的 STM32 单片机入门系列视频（新版）。教程共 54 集，内容涵盖了 STM32 开发环境搭建、GPIO、串口通信、I2C、SPI、中断系统、时钟树、定时器以及 ADC 等核心知识点，适合嵌入式初学者系统学习。

## 目录与内容概览

### 0. 课程介绍
- **P1** 课程介绍

### 1. 准备工作
主要介绍 STM32 的基本概念、开发环境的安装以及芯片引脚分布。

- **P3** 1.1 [准备]STM32的基本信息
- **P4** 1.2 [准备]STM32的引脚分布
