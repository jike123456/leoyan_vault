# 10.4 [ADC]常规单通道转换

## 主要内容
最简单的 ADC 使用方式：单次、单通道、软件触发。

### 配置流程
1.  开启 ADC1 和 GPIOA 时钟。
2.  配置 ADCCLK 分频 (最大 14MHz)。`RCC_ADCCLKConfig(RCC_PCLK2_Div6);` (72/6=12M)。
3.  配置 GPIO (PA0) 为**模拟输入**。
4.  配置 ADC (`ADC_Init`):
    *   Mode: Independent
    *   ScanConvMode: DISABLE
    *   ContinuousConvMode: DISABLE (单次)
    *   Trigger: Software (软件触发)
5.  配置通道：`ADC_RegularChannelConfig(ADC1, ADC_Channel_0, 1, ADC_SampleTime_55Cycles5);`
6.  开启 ADC (`ADC_Cmd`) 并**校准** (`ADC_StartCalibration`).

### 读取数据
```c
ADC_SoftwareStartConvCmd(ADC1, ENABLE); // 触发
while(ADC_GetFlagStatus(ADC1, ADC_FLAG_EOC) == RESET); // 等待转换完成
return ADC_GetConversionValue(ADC1); // 读结果
```
