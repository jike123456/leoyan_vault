# 8.1 [时钟]时钟树

## 主要内容
详解 STM32 复杂的时钟系统 (RCC)。

### 4个主要时钟源
1.  **HSI (High Speed Internal)**：内部 8MHz RC 振荡器 (精度一般)。
2.  **HSE (High Speed External)**：外部 4-16MHz 晶振 (通常 8MHz，精度高)。
3.  **LSI (Low Speed Internal)**：内部 40kHz (供看门狗)。
4.  **LSE (Low Speed External)**：外部 32.768kHz (供 RTC)。

### 系统时钟 (SYSCLK) 流程
*   HSE (8MHz) -> PLL (锁相环倍频 x9) -> 72MHz。
*   SYSCLK = 72MHz。

### 总线时钟
*   **AHB** (HCLK): 72MHz (不分频)。
*   **APB1** (PCLK1): 36MHz (最大 36M，所以 /2)。*(挂载：USART2/3, I2C, SPI2, TIM2/3/4)*
*   **APB2** (PCLK2): 72MHz (最大 72M)。*(挂载：GPIO, USART1, SPI1, ADC, TIM1/8, EXTI/AFIO)*

### MCO (Microcontroller Clock Output)
PA8 引脚可以输出内部时钟供外部测量。
