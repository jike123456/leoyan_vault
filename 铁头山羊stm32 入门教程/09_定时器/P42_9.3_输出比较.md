# 9.3 [定时器]输出比较

## 主要内容
利用定时器的输出比较 (Output Compare) 通道产生 PWM 波形。

### PWM (Pulse Width Modulation)
*   **频率**：由 ARR 决定。
*   **占空比 (Duty)**：由 CCR (Capture Compare Register) 决定。
*   **分辨率**：1 / ARR。

### 工作原理 (PWM模式1)
*   CNT < CCR 时，输出有效电平 (如高电平)。
*   CNT >= CCR 时，输出无效电平 (如低电平)。
*   通过调节 CCR 的值，可以动态改变高电平的时间宽度 (占空比)。

### 呼吸灯实现
在主循环中不断修改 CCR 的值 (从小到大再到小)，LED 亮度就会渐变。
`TIM_SetCompare1(TIM2, Duty);`
