# 5.6 [SPI]数据收发

## 主要内容
编写 SPI 交换字节的函数。

### 代码实现
```c
uint8_t SPI_SwapByte(uint8_t ByteSend) {
    while (SPI_I2S_GetFlagStatus(SPI1, SPI_I2S_FLAG_TXE) == RESET); // 等待发送缓冲空
    SPI_I2S_SendData(SPI1, ByteSend); // 写入数据

    while (SPI_I2S_GetFlagStatus(SPI1, SPI_I2S_FLAG_RXNE) == RESET); // 等待接收缓冲非空
    return SPI_I2S_ReceiveData(SPI1); // 读取数据
}
```
