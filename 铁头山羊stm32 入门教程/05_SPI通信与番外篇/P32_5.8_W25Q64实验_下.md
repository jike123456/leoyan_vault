# 5.8 [SPI]W25Q64实验(下)

## 主要内容
实现 W25Q64 的读写功能。

### 代码逻辑
1.  **Read ID**: SS拉低 -> 发 0x9F -> 交换3个字节 (MID, DID) -> SS拉高。
2.  **Sector Erase**: 写使能 -> SS拉低 -> 发 0x20 -> 发 3字节地址 -> SS拉高 -> 等待 Busy 结束。
3.  **Page Program**: 写使能 -> SS拉低 -> 发 0x02 -> 发 3字节地址 -> 发数据 (最多 256B) -> SS拉高 -> 等待 Busy。
4.  **Read Data**: SS拉低 -> 发 0x03 -> 发 3字节地址 -> 交换读取数据 -> SS拉高。
