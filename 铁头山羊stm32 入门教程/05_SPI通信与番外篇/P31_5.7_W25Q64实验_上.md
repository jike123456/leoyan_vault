# 5.7 [SPI]W25Q64实验(上)

## 主要内容
驱动 W25Q64 Flash 存储芯片。

### W25Q64 简介
*   8MB (64M-bit) SPI Flash。
*   存储架构：Block (块) -> Sector (扇区 4KB) -> Page (页 256B)。
*   **特性**：
    *   写入前必须先**擦除** (Erase)，擦除后数据变为 0xFF。
    *   写入只能将 1 变为 0，不能将 0 变为 1。
    *   写入操作需要**写使能 (Write Enable)**。

### 常用指令
*   **0x06**: Write Enable (写使能)。
*   **0x9F**: Read JEDEC ID (读取厂商和设备 ID)。
*   **0x03**: Read Data (读数据)。
*   **0x02**: Page Program (页编程/写数据)。
*   **0x20**: Sector Erase (扇区擦除 4KB)。
*   **0x05**: Read Status Register-1 (读状态寄存器，判断是否忙 BUSY)。
