# 6.3 [中断]串口中断编程实验

## 主要内容
综合练习：使用串口接收中断控制 LED。

### 实验现象
*   电脑发送 '1'，LED 点亮。
*   电脑发送 '0'，LED 熄灭。

### 关键代码 (ISR)
```c
void USART1_IRQHandler(void) {
    if (USART_GetITStatus(USART1, USART_IT_RXNE) == SET) {
        uint8_t Data = USART_ReceiveData(USART1);
        if (Data == '1') GPIO_ResetBits(GPIOA, GPIO_Pin_0); // LED ON
        else if (Data == '0') GPIO_SetBits(GPIOA, GPIO_Pin_0); // LED OFF
        USART_ClearITPendingBit(USART1, USART_IT_RXNE);
    }
}
```
