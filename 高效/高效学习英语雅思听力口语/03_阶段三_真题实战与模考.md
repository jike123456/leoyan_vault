# 03_阶段三_真题实战与模考 (实战冲刺期)

**目标**：适应考试节奏，听力抓同义替换，口语逻辑清晰。
**周期**：考前 1 个月

## 一、 核心任务 E：听力倍速与同义替换
-   **倍速训练**：平时练习用 **1.2倍速** 做剑桥雅思真题。习惯了快语速，考试时你会觉得录音很慢。
-   **抓同义替换 (Paraphrase)**：
    -   雅思听力考的不是原词，是替换。
    -   听到录音说 "cost a lot of money"，卷子上选 "expensive"。
    -   **建立错题本**：专门记录你没听出来的同义替换对。

## 二、 核心任务 F：口语全真模考
-   **找搭档/AI**：最好找真人练，如果没有，就用 ChatGPT 语音版或者雅思哥 APP 进行模拟。
-   **抗干扰训练**：在稍微有点嘈杂的环境下练习（比如咖啡馆），模拟考场可能的干扰。
-   **Part 3 逻辑**：Part 3 是拉分项。要用**批判性思维**回答。
    -   *结构*：Point (观点) -> Reason (解释) -> Example (举例) -> Counter-argument (反面/让步)。

## 📅 每日执行清单 (考前 30 天)

| 时间块 | 任务内容 | 关键点 |
| :--- | :--- | :--- |
| **上午 (9:00-12:00)**| **全真模考** | 严格按考试时间，先听力，再阅读，再写作。**中途不暂停**。 |
| **下午 (2h)** | **错题分析 + 口语**| 分析听力错题原因。录音练习一套口语真题。 |
| **晚上 (1h)** | **复习语料库** | 快速过一遍听力单词拼写，防止提笔忘字。 |

**阶段通关标准**：模考分数稳定在目标分（或略高 0.5 分）；口语面对陌生话题能不停顿瞎掰 2 分钟。
