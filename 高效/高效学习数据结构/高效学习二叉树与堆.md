# 高效学习二叉树与堆 (Tree & Heap)

## 一、 二叉树 (Binary Tree) —— 递归的温床
**二叉树是突破算法思维的分水岭。**

### 1. 核心思维：递归 (Recursion)
- **不要人脑模拟递归**：超过 3 层人脑就崩了。
- **怎么想**：
    1.  **定义函数功能**：这个函数是干嘛的？（比如：翻转树）。
    2.  **处理当前节点**：通过根节点我要做什么？（交换左右儿子）。
    3.  **相信你的函数**：递归调用左右子树，相信它们能完成任务，不要去管它们怎么完成的。
    4.  **Base Case**：什么时候停？（节点为空）。

### 2. 遍历方式
- **DFS (前中后序)**：
    - 前序：根 -> 左 -> 右
    - 中序：左 -> 根 -> 右 (**BST 的中序是有序数组**)
    - 后序：左 -> 右 -> 根 (自底向上，适合汇总信息)
- **BFS (层序)**：
    - 用队列实现。

### 3. 特殊树：二叉搜索树 (BST)
- **特性**：左 < 根 < 右。
- **作用**：自带排序属性，查找快 (O(logN))。

## 二、 堆 (Heap) / 优先队列 (Priority Queue)
- **本质**：完全二叉树，但不用链表存，用数组存。
- **特性**：
    - **大顶堆**：堆顶是最大值。
    - **小顶堆**：堆顶是最小值。
- **场景**：
    1.  **Top K 问题**：找出前 K 大的元素。
    2.  **流数据中位数**。
