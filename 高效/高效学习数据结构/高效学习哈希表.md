# 高效学习哈希表 (Hash Table)

## 一、 核心本质
- **目的**：**用空间换时间**。
- **能力**：将查找的时间复杂度从 O(N) 降到 **O(1)**。
- **生活类比**：查字典。哈希函数就是那个“拼音索引”，通过索引直接翻到那一页，而不用一页页翻。

## 二、 什么时候用？
**看到题目里出现“计数”、“是否出现过”、“查找元素”这几个词时，第一时间想到哈希表。**

## 三、 两种形式
1.  **HashSet (集合)**：只存 Key。用来去重、判断存在。
2.  **HashMap (映射)**：存 Key-Value 对。用来记录信息（比如：数字 x 出现了 3 次 -> `map.put(x, 3)`）。

## 四、 刷题精选
1.  **1. 两数之和 (Two Sum)**
    - **为什么用哈希？** 找 `target - x` 是否存在，哈希表最快。
2.  **242. 有效的字母异位词**
    - **计数器思想**：用数组或 Map 统计字符出现的次数。
