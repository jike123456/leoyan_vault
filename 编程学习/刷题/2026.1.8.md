## 151.翻转字符串里的单词
```cpp
class Solution {
public:
    void removeExtraSpaces(string& s){
        int slow = 0;
        for(int i=0; i < s.size(); i++){
            if(s[i] != ' '){
                if(slow != 0) s[slow++] = ' ';
                while(i < s.size() && s[i] != ' '){
                    s[slow++] = s[i++];
                }
            }
        }
        s.resize(slow);
    }
    string reverseWords(string s) {
        removeExtraSpaces(s);
        reverse(s.begin(), s.end());
        int start = 0;
        for(int i = 0; i <= s.size(); i++){
            if(i == s.size() || s[i] == ' '){
                reverse(s.begin()+start, s.begin()+i);
                start = i+1;
            }
        }
        return s;
    }
};
```
空格（单词）、空格（单词）；除了在slow =1的时候。

要注意`i<s.size()` i++的时候很容易出现越界，得放在前面。

`reverse(start,end)` 是左开右闭，不需要-1。
