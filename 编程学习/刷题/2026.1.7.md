## 344. 反转字符串
定义应该`temp`
```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        int left = 0, right = s.size()-1;
        while(left < right){
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            left++;
            right--;
        }
    }
};
```

## 541.反转字符串II
`for(int i = 0; i < s.size(); i++)`:习惯性i++；可以进行==变通的==；`i += 2k`

reverse(起始位置迭代器, 结束位置迭代器);
``reverse(s.begin() + i, s.begin() + i + k);``

- **`s.begin()`**：这就相当于告诉编译器，“我要在这个叫 **`s`** 的字符串里找位置”。
- **`+ i`**：这是偏移量。
- **合起来**：`s.begin() + i` 就是一个**指向 `s` 字符串内部第 `i` 个字符的“指针”**（迭代器）。
```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        for (int i = 0; i < s.size(); i += (2 * k)) {
            // 1. 每隔 2k 个字符的前 k 个字符进行反转
            // 2. 剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符
            if (i + k <= s.size()) {
                reverse(s.begin() + i, s.begin() + i + k );
            } else {
                // 3. 剩余字符少于 k 个，则将剩余字符全部反转。
                reverse(s.begin() + i, s.end());
            }
        }
        return s;
    }
};
```

==`2*k`而非==`2k`

## 54.替换数字
从后往前：后面的只需要挪动一次。

从前往后：后面的前面移动，它就得一直往后移动。

```cpp
if(s[i] >= '0' && s[i] <= '9'){
                count++;
            }
```
asci码。
```cpp
#include <iostream>
using namespace std;
int main() {
    string s;
    while (cin >> s) {
        int sOldIndex = s.size() - 1;
        int count = 0; // 统计数字的个数
        for (int i = 0; i < s.size(); i++) {
            if (s[i] >= '0' && s[i] <= '9') {
                count++;
            }
        }
        // 扩充字符串s的大小，也就是将每个数字替换成"number"之后的大小
        s.resize(s.size() + count * 5);
        int sNewIndex = s.size() - 1;
        // 从后往前将数字替换为"number"
        while (sOldIndex >= 0) {
            if (s[sOldIndex] >= '0' && s[sOldIndex] <= '9') {
                s[sNewIndex--] = 'r';
                s[sNewIndex--] = 'e';
                s[sNewIndex--] = 'b';
                s[sNewIndex--] = 'm';
                s[sNewIndex--] = 'u';
                s[sNewIndex--] = 'n';
            } else {
                s[sNewIndex--] = s[sOldIndex];
            }
            sOldIndex--;
        }
        cout << s << endl;       
    }
}
```

`sOldIndex` `sNewIndex` 新指针与老指针，一个指向新的结尾，一个指向老结尾；

遍历计数`count` 数字`0` 到`9` ;如果有就计数，最后用`.resize(s.size() +count*5)` 更新空间。

由老指针开始向前走，走到下标<0 停止。`while (sOldIndex >= 0)`

如果是数字，则对新指针进行写入`s[sNewIndex--] = 'r';......` 如果是字母则是将老指针的值赋值给新指针`s[sNewIndex--] = s[sOldIndex];`

最后老指针往前走，继续下一个循环。
