## 344. 反转字符串
定义应该`temp`
```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        int left = 0, right = s.size()-1;
        while(left < right){
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            left++;
            right--;
        }
    }
};
```

```cpp
	class Solution {
	public:
	    void reverseString(vector<char>& s) {
	        for (int i = 0, j = s.size() - 1; i < s.size()/2; i++, j--) {
	            swap(s[i],s[j]);
	        }
	    }
	};
```

## 541.反转字符串II
`for(int i = 0; i < s.size(); i++)`:习惯性i++；可以进行==变通的==；`i += 2k`

reverse(起始位置迭代器, 结束位置迭代器);
``reverse(s.begin() + i, s.begin() + i + k);``

- **`s.begin()`**：这就相当于告诉编译器，“我要在这个叫 **`s`** 的字符串里找位置”。
- **`+ i`**：这是偏移量。
- **合起来**：`s.begin() + i` 就是一个**指向 `s` 字符串内部第 `i` 个字符的“指针”**（迭代器）。
```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        for (int i = 0; i < s.size(); i += (2 * k)) {
            // 1. 每隔 2k 个字符的前 k 个字符进行反转
            // 2. 剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符
            if (i + k <= s.size()) {
                reverse(s.begin() + i, s.begin() + i + k );
            } else {
                // 3. 剩余字符少于 k 个，则将剩余字符全部反转。
                reverse(s.begin() + i, s.end());
            }
        }
        return s;
    }
};
```

==`2*k`而非==`2k`
