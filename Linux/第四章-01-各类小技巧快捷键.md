# Linux 各类小技巧与快捷键

掌握Linux的快捷键和一些实用技巧能够极大地提升在命令行下的工作效率。本笔记将分类介绍一些最常用的快捷键和命令行技巧。

---

## 一、终端快捷键

这些快捷键主要用于在终端（Terminal）中进行快速的光标移动、文本编辑和进程控制。

### 1. 光标移动

| 快捷键        | 功能描述           |
| :--------- | :------------- |
| `Ctrl + A` | 移动光标到当前行的开头。   |
| `Ctrl + E` | 移动光标到当前行的末尾。   |
| `Alt + B`  | 光标向后（左）移动一个单词。 |
| `Alt + F`  | 光标向前（右）移动一个单词。 |
| `Ctrl + ←` | 移动到上一个单词的开头。   |
| `Ctrl + →` | 移动到下一个单词的开头。   |

### 2. 文本编辑

| 快捷键         | 功能描述                                  |
| :---------- | :------------------------------------ |
| `Ctrl + U`  | **剪切**从光标位置到行首的所有字符。                  |
| `Ctrl + K`  | **剪切**从光标位置到行尾的所有字符。                  |
| `Ctrl + W`  | **剪切**光标前的一个单词。                       |
| `Ctrl + Y`  | **粘贴**使用`Ctrl+U`, `K`, `W`剪切的文本。      |
| `Ctrl + D`  | 删除光标所在位置的字符。如果当前行为空，则会关闭终端会话（logout）。 |
| `Backspace` | 删除光标前的一个字符。                           |

### 3. 命令历史

| 快捷键        | 功能描述                                 |
| :--------- | :----------------------------------- |
| `↑` / `↓`  | 上下翻阅历史命令。                            |
| `Ctrl + R` | **反向搜索**历史命令。输入关键字会实时匹配最近的命令，按回车执行。  |
| `!!`       | 执行上一条命令。                             |
| `!string`  | 执行历史记录中最近一条以`string`开头的命令。例如 `!vim`。 |
|            |                                      |

### 4. 终端控制

| 快捷键 | 功能描述 |
| :--- | :--- |
| `Tab` | **命令和文件名自动补全**。连按两次`Tab`可以显示所有匹配项。 |
| `Ctrl + L` | **清屏**，相当于执行`clear`命令。 |
| `Ctrl + C` | **终止**当前正在前台运行的命令或进程。 |
| `Ctrl + Z` | **挂起**当前命令，将其暂停并放入后台。可以使用`fg`恢复。 |
| `Ctrl + S` | 锁定终端屏幕，阻止屏幕输出。 |
| `Ctrl + Q` | 解锁终端屏幕，恢复屏幕输出。 |
| `Ctrl + Shift + T` | 在当前终端窗口中新建一个标签页。 |
| `Ctrl + Shift + W` | 关闭当前的标签页。 |

## 二、实用命令行技巧

### 1. 命令组合与重定向

- **管道符 `|`**
  **理论**：将前一个命令的标准输出（stdout）作为后一个命令的标准输入（stdin）。这是Linux命令组合的核心。
  **示例**：查看当前目录下的文件，并筛选出其中包含 "test" 的文件。
  ```bash
  ls -l | grep "test"
  ```

- **输出重定向 `>` 和 `>>`**
  **理论**：
    - `>`：将命令的输出写入到一个文件中，如果文件已存在，会**覆盖**原有内容。
    - `>>`：将命令的输出**追加**到一个文件的末尾，不会覆盖原有内容。
  **示例**：
  ```bash
  # 将 "Hello World" 写入 file.txt，覆盖原有内容
  echo "Hello World" > file.txt

  # 将日期追加到 file.txt 的末尾
  date >> file.txt
  ```

### 2. 常用高效命令

- **`sudo !!`**
  **理论**：当你执行一条命令后，系统提示“权限不足”（permission denied）时，这个技巧非常有用。`!!`代表上一条命令，`sudo !!`就是以管理员权限重新执行上一条命令。
  **示例**：
  ```bash
  # 尝试安装软件，但忘记加 sudo
  apt-get update
  # > permission denied...

  # 使用 sudo 重新执行上一条命令
  sudo !!
  # > sudo apt-get update
  ```

- **`history`**
  **理论**：显示在当前会话中执行过的所有命令的历史记录。
  **示例**：
  ```bash
  history
  ```

- **`alias`**
  **理论**：为常用的、复杂的命令创建别名，简化操作。可以将 `alias` 命令添加到 `~/.bashrc` 或 `~/.zshrc` 文件中使其永久生效。
  **示例**：创建一个 `ll` 的别名来代替 `ls -alF`。
  ```bash
  alias ll='ls -alF'

  # 现在可以直接使用 ll
  ll
  ```

- **`tail -f`**
  **理论**：实时监控文件的末尾追加内容，常用于查看日志文件。
  **示例**：实时查看系统日志。
  ```bash
  tail -f /var/log/syslog
  ```
