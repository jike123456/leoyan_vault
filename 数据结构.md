# 链表：寻宝游戏（火车）
东西不在一起，通过指针进行连接
火车头：头节点；指针

## 指针：连接（找到其他线索）
不是宝藏，是线索，是连接火车的。
## 变量
是一个宝藏；火车的车厢
通过指针去找宝藏。

* cpp示例
	```
	#include <iostream>
	using namespace std;
	
	int main() {
	    int treasureBox = 100; // 📦 宝箱里有 100 个金币
	    
	    // ⭐ 定义指针：注意这个星星 *
	    // map 是藏宝图，&treasureBox 意思是“获取宝箱的地址”
	    int* map = &treasureBox; 
	
	    cout << "宝箱里的金币: " << treasureBox << endl;
	    
	    // 🕵️ 使用指针：*map 意思是“去地图指的地方看看”
	    // 我们通过地图，把那个位置的金币改成了 9999
	    *map = 9999; 
	
	    cout << "现在宝箱里的金币: " << treasureBox << endl;
	    // 即使我们没直接碰 treasureBox，它也变了！
	    
	    return 0;
	}
	```
