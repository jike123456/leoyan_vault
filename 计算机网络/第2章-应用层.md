# 第2章 应用层 (Application Layer)

本章是自顶向下方法的第一站，详细探讨了网络应用程序以及它们所使用的应用层协议。

## 核心内容

*   **网络应用程序原理 (Principles of Network Applications)**
    *   网络应用体系结构：客户机-服务器 (C/S) 结构、对等 (P2P) 结构。
    *   进程通信：进程如何通过网络相互通信。
    *   运输层服务模型：应用层需要运输层提供什么样的服务 (TCP vs UDP)。
*   **Web 和 HTTP (HyperText Transfer Protocol)**
    *   HTTP协议概览：非持续连接与持续连接。
    *   HTTP报文格式：请求报文和响应报文。
    *   Cookie、Web缓存（代理服务器）。
*   **电子邮件 (SMTP, POP3, IMAP)**
    *   SMTP (Simple Mail Transfer Protocol)：邮件发送协议。
    *   POP3 (Post Office Protocol version 3) 和 IMAP (Internet Message Access Protocol)：邮件接收协议。
*   **DNS (Domain Name System)**
    *   DNS提供的服务：域名到IP地址的转换。
    *   DNS工作原理：分布式、层次化的数据库。
    *   DNS记录和报文。
*   **P2P 应用 (Peer-to-Peer Applications)**
    *   文件分发，如BitTorrent。
*   **视频流和内容分发网 (CDN)**
    *   HTTP流、DASH流。
    *   CDN (Content Distribution Network) 的原理。
*   **Socket 编程**
    *   使用UDP和TCP进行Socket编程的介绍。

本章通过具体的应用层协议实例，展示了网络应用程序是如何构建和工作的，是理解网络服务的基础。
