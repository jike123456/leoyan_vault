# 网络应用程序原理

- **网络应用体系结构**
  - 客户机-服务器 (Client-Server) 结构
  - 对等 (Peer-to-Peer, P2P) 结构
  - 混合结构 (Hybrid)
- **进程通信**
  - 客户端进程与服务器进程
  - 进程与计算机网络之间的接口：套接字 (Socket)
- **运输服务**
  - 可靠数据传输 (Reliable Data Transfer)
  - 吞吐量 (Throughput)
  - 定时 (Timing)
  - 安全 (Security)
- **因特网提供的运输服务**
  - TCP 服务：面向连接、可靠数据传输、拥塞控制
  - UDP 服务：无连接、不可靠数据传输
- **应用层协议**
  - 定义：报文类型、语法、语义、规则
