# Web 和 HTTP

- **HTTP 概览**
  - HTTP (HyperText Transfer Protocol) 作为Web的应用层协议
  - 客户-服务器模型
- **连接类型**
  - 非持续连接 (Non-Persistent Connection)
  - 持续连接 (Persistent Connection)
- **HTTP 报文格式**
  - 请求报文 (Request Message): 请求行、首部行、实体主体
  - 响应报文 (Response Message): 状态行、首部行、实体主体
- **用户与服务器的交互**
  - Cookie: 用于状态管理
- **Web 缓存 (代理服务器)**
  - 目标：减少响应时间、减少网络流量
  - 条件GET方法
