# 电子邮件协议

- **因特网电子邮件系统组件**
  - 用户代理 (User Agents)
  - 邮件服务器 (Mail Servers)
  - 简单邮件传输协议 (Simple Mail Transfer Protocol, SMTP)
- **SMTP**
  - 作为邮件发送的“推”协议
  - 与HTTP的对比
- **邮件访问协议**
  - POP3 (Post Office Protocol, version 3): “拉”协议，用于下载邮件
  - IMAP (Internet Message Access Protocol): 更复杂的邮件访问协议，允许在服务器上管理邮件
  - 基于Web的电子邮件
