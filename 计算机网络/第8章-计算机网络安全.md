# 第8章 计算机网络安全 (Security in Computer Networks)

随着网络应用的普及，网络安全变得至关重要。本章介绍了网络安全的基本原则、常见的威胁以及用于构建安全网络应用的密码学和网络安全协议。

## 核心内容

*   **什么是网络安全？**
    *   安全属性：机密性 (Confidentiality)、认证 (Authentication)、报文完整性 (Message Integrity)、可用性 (Availability)。
    *   常见的威胁：恶意软件、拒绝服务攻击 (DoS)、嗅探、欺骗。
*   **密码学原理**
    *   对称密钥加密 (Symmetric Key Cryptography)：DES, AES。
    *   公开密钥加密 (Public Key Cryptography)：RSA。
*   **报文完整性与数字签名**
    *   密码散列函数 (Cryptographic Hash Functions)：MD5, SHA-1, SHA-256。
    *   报文认证码 (Message Authentication Code, MAC)。
    *   数字签名 (Digital Signatures)。
*   **端点认证 (End-Point Authentication)**
    *   认证协议的设计与原理。
*   **安全电子邮件**
    *   PGP (Pretty Good Privacy)。
*   **确保TCP连接的安全：SSL/TLS**
    *   安全套接字层 (Secure Sockets Layer, SSL) 和传输层安全 (Transport Layer Security, TLS) 的目标和工作原理。
    *   握手过程。
*   **网络层安全：IPsec**
    *   IPsec提供的服务。
    *   隧道模式和传输模式。
*   **确保无线局域网的安全**
    *   有线等效保密 (WEP)。
    *   WiFi保护访问 (WPA, WPA2, WPA3)。
*   **防火墙和入侵检测系统 (IDS)**
    *   防火墙：传统分组过滤器、状态分组过滤器、应用网关。
    *   入侵检测系统 (Intrusion Detection Systems)。

本章是理解现代网络安全体系的入门，内容涵盖了从基本密码学工具到复杂的安全协议，为构建和维护安全的网络环境提供了理论基础。
