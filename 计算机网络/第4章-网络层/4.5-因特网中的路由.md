# 因特网中的路由

- **自治系统 (Autonomous Systems, AS)**
  - 在相同管理控制下的路由器集合。
  - 每个AS有唯一的AS号 (ASN)。
- **内部网关协议 (Interior Gateway Protocols, IGP)**
  - 在同一个AS内部使用的路由协议。
  - **RIP (Routing Information Protocol)**: 一种经典的DV协议。
  - **OSPF (Open Shortest Path First)**: 一种广泛使用的LS协议。
- **外部网关协议 (Exterior Gateway Protocols, EGP)**
  - 用于在不同的AS之间交换路由信息。
  - **BGP (Border Gateway Protocol)**: 当前因特网事实上的标准EGP。
    - BGP是“路径向量”协议。
    - BGP不仅仅是寻找最短路径，更重要的是实施策略。
    - BGP报文和路由选择。
- **广播和多播路由**
  - **广播**: 将分组发送给网络中的所有主机。
  - **多播**: 将分组发送给一个特定的主机组。
    - 多播路由算法。
