# 报文完整性与数字签名

- **密码散列函数 (Cryptographic Hash Functions)**
  - 将任意长度的报文映射为固定长度的摘要。
  - 特性：单向性、抗碰撞性。
  - 算法：MD5, SHA-1, SHA-256。
- **报文认证码 (Message Authentication Code, MAC)**
  - 使用共享密钥和散列函数来生成一个认证码，附加在报文后。
  - 用于验证报文的完整性和来源。
- **数字签名 (Digital Signatures)**
  - 使用发送方的私钥对报文的散列值进行加密。
  - 接收方使用发送方的公钥解密，以验证报文的来源、完整性和不可否认性。
- **公钥基础设施 (Public Key Infrastructure, PKI)**
  - 用于管理公钥的分发和认证。
  - **证书颁发机构 (Certificate Authority, CA)**: 负责签发数字证书，将公钥与特定实体绑定。
